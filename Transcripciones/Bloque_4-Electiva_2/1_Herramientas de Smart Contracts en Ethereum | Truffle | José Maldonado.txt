Una de las herramientas más utilizadas dentro del desarrollo de Ethereum y de redes compatibles
con la EVM es Truffle Suite. Truffle Suite es en realidad una suite de tres herramientas
que son conocidas como Truffle, que es un sistema para desarrollar y crear entornos
de desarrollo para aplicaciones descentralizadas y Smart Contracts. Ganache, que es una aplicación
que nos permite crear una blockchain dentro de nuestro computador y que explicaremos más
adelante cómo instalarla y cómo configurarla para que puedas interactuar con ella utilizando
una Metamax dentro de tu computador. Y Driftlet, que es simplemente una serie de herramientas
que nos permiten crear frontends y manejo de data para nuestras aplicaciones descentralizadas.
Estas tres herramientas son las que conforman todo lo que sería la suite de Truffle. En
este caso, vamos a ver primero la red de Truffle, lo que sería Truffle como tal. Vamos a ir
a su GitHub y desde allí podemos ver que es completamente código libre, software libre.
Podemos llegar a instalarla simplemente utilizando este comando, un comando de npm install de
forma global Truffle. En este caso, le recordamos que deben tener instalado las últimas versiones
de Node, deben ser posibles las versiones LTS o utilizar un instalador del tipo nvm
dentro de lo que serían sus computadores. Y con ello podemos llegar y empezar a generar
todo lo que serían las interfaces iniciales de desarrollo de nuestro Smart Contract. ¿Qué
podemos crear entonces con Truffle? Bueno, con Truffle podemos crear desde token ERC20
hasta NFT o aplicaciones descentralizadas de cualquier tipo. Podemos llegar y crear
entornos de desarrollo que nos permitan interactuar con IPFS, que en este caso es el sistema de
almacenamiento descentralizado que utilizan muchas aplicaciones descentralizadas. O podemos
simplemente crear un sistema para mintear NFTs, nuestras propias conexiones NFTs directamente
dentro de lo que sería la red Ethereum o dentro de las redes que estemos desarrollando
que estén dentro de la computabilidad de Truffle. Ahora bien, GANache, por otro lado,
es simplemente un software que nos permite llegar y instalar todo lo que sería todo
lo necesario para llegar y poner a funcionar una blockchain virtual dentro de nuestro computador.
Esta blockchain virtual es completamente compatible con Ethereum, de forma que si desarrollas
para esta blockchain virtual, para esta blockchain que genera GANache, puedes simplemente tomar
tu código y llevarla a la red de Ethereum sin mayores problemas porque va a funcionar sin ninguna
dificultad, ya que es completamente compatible y de hecho GANache tiene la capacidad de poder
llegar y configurarse para diferentes hard forks que están dentro de lo que sería el código de
Ethereum. Es fácil instalarlo, simplemente seleccionas el tipo de sistema operativo que
estás utilizando. En este caso nos aparece la instalación para Windows porque estamos
en sistema operativo Windows, pero si quieres para otros sistemas operativos, puedes simplemente
hacer clic en este enlace y rápidamente puedes ver aquí las diferentes versiones que aparecen
disponibles, que en este caso son para Linux en formato AppImage, para Mac en formato DMG y
también para Windows en formato ZDub y AppX. Esto significa que puedes instalar GANache
tanto para los tres sistemas, en este caso que serían Windows, Linux y MacOS. No hay ningún tipo
de inconveniente y vas a tener las mismas funcionalidades en cada uno de estos casos.
Finalmente Drizztle es simplemente una colección de librerías que nos permiten llegar y manejar
data a nivel de frontend que nos facilita la interacción con los usuarios y la compartición
de data entre las diferentes Dapps que estemos realizando, las diferentes partes de las Dapps
que estamos realizando. Esto es un uso bastante más específico, pero que realmente tiene su utilidad
cuando hablamos de aplicaciones un poco más complejas, puede ser por ejemplo la construcción
de un mercado del tipo OpenSea o RedLibre, en el que este tipo de funciones serían excepcionalmente
útiles para llegar y crear una mejor interacción con los usuarios. Entonces como pueden ver,
en este caso TrueFace Suite es un compendio de herramientas que nos resultan muy útiles y que
facilitan mucho el desarrollo de las aplicaciones descentralizadas. No solamente porque contienen
ya esqueletos o bases ya predefinidas para construir estas aplicaciones, sino porque llegan
y tienen una serie de documentación que facilita mucho este desarrollo. Así que por ello comenzaremos
a estudiar un poco esta suite y cómo podemos aprovecharla al máximo para llegar a crear
nuestras aplicaciones descentralizadas sin mayores problemas.
